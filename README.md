# Clasificador-de-Im-genes-con-CNN---CIFAR-10
Proyecto desarrollado en Python utilizando TensorFlow y Keras para la clasificación automática de imágenes mediante una Red Neuronal Convolucional (CNN).  El modelo fue entrenado utilizando el conjunto de datos CIFAR-10, compuesto por 60.000 imágenes a color de 32x32 píxeles distribuidas en 10 categorías diferentes.
